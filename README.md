Requisitos
----------

- curl
- grep, cat...

Como usar
---------

### Ver titulos das propostas

	cd fisl-segue-cache
	. bin/list_titles.sh

### Ver uma proposta

	cat data/[numero]

### Baixar novas propostas ou atualizar existentes

	. bin/get_page.sh [numero_da_pagina]

### Busca

	. bin/search.sh [keyword]

### Listar autores

	. bin/list_authors.sh
